
### Get started

- Clone project
  ```
  git clone https://github.com/umershaikh123/DiscordTicketingBot.git
  ```
- change directory
- install packages
  ```
  npm install  // use node version greater than 18
  ```
- set env
  ```
  DISCORD_TOKEN=
  KV_URL= 
  KV_REST_API_READ_ONLY_TOKEN= 
  KV_REST_API_TOKEN= 
  KV_REST_API_URL= 
  ```
- Run project
  ```
  node ./index.js
  ```
